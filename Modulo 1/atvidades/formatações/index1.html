Principais Formatações

Negrito/Destaque

Nesta frase temos um termo em destaque usando a tag <strong> (semantica).

Nesta frase temos uma palavara em negrito usando a tag <b> (não semantica).

Itálico/Ênfase

Nesta frase temos um termo em ênfase usando a tag <em> (semantica).

Nesta frase temo um termo em italico usando a teg <i> (não semantica).

Texto Marcado

Podemos marcar texto usando a tag <mark>

Para trocarmos a cor do marcador de texto temos qeu adicionar na tag <head> uma outra tag chamada <style> como se fosse uma "variavel global" para que a cor do marcador mude.

Texto Excluido

Podemos marcar um texto como excluido usando a tag <del>.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e dizer que o mesmo foi adicionado depois, usando a tag <ins>.

Texto Sobrescrito

Para inserir coisas do tipo x20-3. Usando a tag <sup>.

Texto Subscrito

Para inserir coisas do tipo H20. Usando a tag <sub>.
